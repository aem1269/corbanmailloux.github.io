alittleteapot.im is a beautifully animated teapot, made with CSS animations.

Just to get the lettering right, it uses animate.css, lettering.js, and textillate.js.

The teapot animation, on the other hand, was made by me.

The result is available at alittleteapot.im, and the code is available on GitHub.

Any questions can be directed at alittleteapot.im@corb.co.
